3) Quais outros protocolos de comunicação para troca de mensagens via internet existem? Pesquise e cite 3 exemplos, descrevendo sua função e aplicação.

  • PROTOCOLO 1: FTP - File Tranfer Protocol. Serve para realizar transferências de arquivos de um servidor FTP.
    O FTP permite modificar diretórios, inserir e apagar arquivos no servidor.

  • PROTOCOLO 2: SSL - Secure Socket Layer. Permite estabelecer conexões seguras entre client e server.
    O SSL utiliza criptografia tranferir dados de forma segura.
    Aparece na forma de https://

  • PROTOCOLO 3: SSH - Secure Shell. Permite acesar outro computador remotamente com uma espécie de promtp de comando.
    A diferença é que todos os textos são encriptados.

5) Pequisem sobre as limitações do HTTP 1.1 e como o HTTP2 vence essa limitação, cite 2 dessas limitações e suas respostas no HTTP2

RESPOSTA:
O HTTP 1.1 limita a quantidade de downloads simultâneos (entre 4 e 8, no Chrome); O HTTP2 usa Multiplexing, que realiza os donwloads assincronamente, sem ter que esperar a resposta dos requests pendentes
O HTTP 1.1 não prioriza o render, baixando tudo em ordem; O HTTP2 envia todos os requests ao mesmo tempo e define o que é mais interessante carregar primeiro.
